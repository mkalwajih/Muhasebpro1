Flutter Web Project

This is a Flutter application configured for web deployment. The project source code is ready and available in this repository.

Project Structure

  • lib/ - Dart source code
  • web/ - Web platform files
  • assets/ - Project assets
  • pubspec.yaml - Dependencies

To Build & Run

  • Install Flutter SDK locally
  • Run flutter pub get
  • Run flutter build web
  • Deploy the build/web folder
